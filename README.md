# Fiat500 Car Price Prediction

## 📌 Overview

This project predicts the selling price of used Fiat 500 cars using Machine Learning regression techniques. It demonstrates data preprocessing, feature selection, model training, and regression model evaluation.

---

## 🎯 Objective

Build a regression model to estimate the selling price of a used Fiat 500 based on vehicle features.

---

## 📂 Dataset

- **Dataset:** Fiat 500 Used Car Dataset
- **Problem Type:** Regression
- **Target Variable:** Price

---

## 🤖 Machine Learning Model

- Linear Regression

---

## ⚙️ Data Preprocessing

- Handling missing values
- Feature selection
- Data cleaning
- Train-Test Split

---
